# Linux Log Monitor

A baah script that analyzes Linux Log files and generates a summary report.

## Features

- Counts INFO messages
- Counts WARNING messages
- Counts ERROR messages
- Counts FAILED events
- Displays the current date
- Generates a formatted report

## Commands Used

- grep
- echo
- date
- chmod
